# 本地图片浏览器

一个基于HTML、CSS和JavaScript的现代化本地图片浏览应用，支持多种查看模式和智能的图片管理功能。

## ✨ 功能特性

### 🖼️ 多种浏览模式
- **瀑布流模式**：优雅的卡片式布局，适合大量图片的快速浏览
- **漫画模式**：专为漫画阅读优化的垂直连续视图
- **文件夹模式**：按文件夹组织图片，便于管理

### 🚀 性能优化
- **无限滚动**：流畅的无限滚动加载，支持一次性加载大量图片
- **懒加载机制**：智能图片懒加载，优化内存使用
- **内存管理**：自动清理未使用的图片缓存，防止内存溢出
- **性能监控**：实时显示图片数量和内存使用情况

### 🔍 便捷功能
- **文件夹浏览**：支持选择本地文件夹进行图片浏览
- **搜索过滤**：快速查找特定图片
- **面包屑导航**：清晰的目录结构显示
- **全屏查看**：沉浸式图片查看体验
- **缩放功能**：支持图片缩放和平移
- **键盘快捷键**：键盘导航支持

### 💫 用户体验
- **响应式设计**：适配不同屏幕尺寸
- **平滑动画**：流畅的过渡效果
- **现代化界面**：简洁优雅的用户界面
- **即时反馈**：实时状态显示和操作提示

## 🚀 使用方法

1. **打开应用**：直接在浏览器中打开`本地图片浏览器.html`文件
2. **选择文件夹**：点击"选择文件夹"按钮，选择包含图片的本地文件夹
3. **浏览图片**：应用会自动扫描并显示文件夹中的所有图片
4. **切换模式**：使用顶部的模式按钮切换不同的浏览方式
5. **查看详情**：点击图片进入全屏模式，使用鼠标滚轮缩放

## 🎯 支持的图片格式

- JPEG (.jpg, .jpeg)
- PNG (.png)
- GIF (.gif)
- WebP (.webp)
- BMP (.bmp)
- SVG (.svg)

## 🔧 技术特点

### 核心技术
- **原生JavaScript**：无需额外依赖，纯原生JS实现
- **现代CSS**：使用CSS Grid、Flexbox等现代布局技术
- **HTML5 API**：利用File System API进行文件访问

### 性能优化
- **智能缓存**：URL缓存机制避免重复创建对象URL
- **懒加载Observer**：使用Intersection Observer实现高效的懒加载
- **内存监控**：实时监控内存使用，自动清理资源
- **防抖节流**：优化滚动和缩放性能

### 架构设计
- **模块化结构**：清晰的功能模块划分
- **状态管理**：统一的状态管理机制
- **事件驱动**：基于事件的交互设计

## 🎮 快捷键操作

### 全屏模式
- `←` / `→`：切换上一张/下一张图片
- `Esc`：退出全屏模式
- `鼠标滚轮`：缩放图片
- `鼠标拖拽`：平移缩放后的图片

### 导航操作
- `回到顶部`按钮：快速返回页面顶部

## 💡 使用提示

1. **首次加载**：首次选择文件夹时可能需要一些时间来扫描图片
2. **大量图片**：应用支持同时浏览数千张图片，会自动进行性能优化
3. **内存管理**：当检测到内存使用过高时，应用会自动清理未使用的缓存
4. **文件夹限制**：建议单个文件夹图片数量控制在5000张以内以获得最佳性能
